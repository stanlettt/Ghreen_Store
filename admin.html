Connexion Google Sheets

Connectez un Google Sheet pour sauvegarder vos produits en ligne. Ainsi, même si vous videz le cache de votre navigateur, vos produits sont toujours disponibles.

1
Créez un Google Sheet sur sheets.google.com et nommez-le Ghreen_Store_Produits
2
Ajoutez ces colonnes en ligne 1 :
id  |  nom  |  cat  |  prix  |  badge  |  note  |  icone  |  photos
3
Publiez le Sheet : Fichier → Partager → Publier sur le web → Feuille 1 → Format CSV → Publier. Copiez le lien.
4
Collez le lien CSV ici :
5
Exportez vos produits actuels vers Sheets en cliquant sur l'onglet "Exporter".
